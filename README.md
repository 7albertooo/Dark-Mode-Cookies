# Dark-Mode-Cookies

Esto es un proyecto simple para practicar las Cookies en PHP, el método GET y estilos Dark en Tailwind CSS, el cual permite a los usuarios alternar entre el modo oscuro y el modo claro en un sitio web. La preferencia del usuario se guarda en una cookie para que se mantenga incluso después de cerrar el navegador.

# Tecnologías Utilizadas
- PHP
- Tailwind CSS
- HTML

# Cómo Usar
1. Clona este repositorio en tu máquina local.
2. Asegúrate de tener un servidor web con soporte para PHP como XAMPP.
3. Coloca los archivos del proyecto en el directorio raíz de tu servidor web.
4. Instala Tailwind CSS siguiendo las instrucciones oficiales en [Tailwind CSS Installation](https://tailwindcss.com/docs/installation).

# Instrucciones para Ejecutar el Proyecto
4. Abre tu navegador web y navega a `http://localhost/nombre-del-proyecto` (reemplaza `nombre-del-proyecto` con el nombre de la carpeta donde colocaste los archivos).
5. Haz clic en el botón "Toggle Dark Mode" para cambiar entre el modo oscuro y el modo claro. La preferencia se guardará en una cookie.
